# DC Circuit Lab

An interactive, single-file web app for learning DC circuit analysis —
series, parallel, and combination circuits. Includes a live simulator,
practice problems, a printable worksheet generator, and a circuit builder.

**Live app:** https://tflaris.github.io/dc-circuit-lab/

Customized for the Electrical Training Alliance – Jacksonville
(IBEW–NECA apprenticeship program).

## Features

- **Simulator** — set source voltage and resistor values; view the schematic and calculated values (E, I, R, P)
- **Practice** — generate and solve problems, with printable output
- **Worksheet** — build printable worksheets with problem and solution modes
- **Circuit Builder (beta)** — place components and validate/reduce circuits

## Usage

Open the live link above in any modern browser — no install required.
Works offline once loaded, and can be added to a phone's home screen.
